\begin{zremark}
  对定理3.4直观版本：

  定理 3.4$^\prime$： 若$\lim\limits_{x \to x_0} f(x) = A$，那么对任意实数
  $B < A < C$，存在$\delta > 0$，当$0 < |x - x_0| < \delta$时，有
  \begin{align*}
    B < f(x) < C
  \end{align*}
\end{zremark}

\textbf{证明：}

$\lim\limits_{x \to x_0} f(x) = A$，由定义
对$\epsilon = min\{A - B, C - A\}$，$\exists \delta > 0$，
当$0 < |x - x_0| < \delta$时，有
\begin{align*}
  |f(x) - A| < \epsilon \\
  B \leq A - \epsilon < f(x) < A + \epsilon \leq C
\end{align*}
